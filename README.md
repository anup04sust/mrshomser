# 🤖 mrSomsher (সবজান্তা শমসের)

> A playful yet powerful AI assistant that feels like your smartest (and
> slightly overconfident) friend.

------------------------------------------------------------------------

## 🧠 Project Vision

mrSomsher is a self-hosted AI assistant built on top of local LLMs (via
Ollama), designed to provide:

-   Practical answers\
-   Real-world problem solving\
-   Bangla + English hybrid communication\
-   A strong personality (not boring AI)

------------------------------------------------------------------------

## 😏 Core Idea

"সবজান্তা শমসের" plays on the idea of a "know-it-all".

> Doesn't know everything --- but will figure it out.

## 🧱 Tech Stack

-   **Frontend:** Next.js 16, React 19, TailwindCSS
-   **Backend:** Next.js API Routes, Node.js 20
-   **Database:** MongoDB
-   **AI:** Ollama (qwen3.5:4b model)
-   **Auth:** JWT + bcrypt
-   **Dev Environment:** DDEV + Docker

------------------------------------------------------------------------

## 🚀 Quick Start

1. **Prerequisites:** DDEV, Docker, pnpm

2. **Clone & Start:**
   ```bash
   git clone https://github.com/yourusername/mrshomser.git
   cd mrshomser
   ddev start
   ```

3. **Pull AI Model:**
   ```bash
   ddev exec -s ollama "ollama pull qwen3.5:4b"
   ```

4. **Access:** https://mrshomser.ddev.site
