## 🚀 Upload Instructions

### Method 1: Automated Script (Recommended)

```bash
# Run the upload script
./upload-wiki.sh
```

The script will:
1. Clone your wiki repository
2. Copy all markdown files
3. Commit changes
4. Push to GitHub
5. Clean up temporary files

### Method 2: Manual Upload (Web Interface)

1. Go to https://github.com/anup04sust/mrshomser/wiki
2. Click "Create the first page" or "New Page"
3. For each `.md` file:
   - Copy filename (without .md) as page title
   - Paste markdown content
   - Save page

**Files to upload:**
```
Home.md                 → "Home"
Installation-Guide.md   → "Installation Guide"
API-Reference.md        → "API Reference"
Architecture.md         → "Architecture"
Troubleshooting.md      → "Troubleshooting"
FAQ.md                  → "FAQ"
Model-Selection.md      → "Model Selection"
```

### Method 3: Git Clone (Advanced)

```bash
# Clone wiki repo
git clone https://github.com/anup04sust/mrshomser.wiki.git

# Copy files
cp wiki/*.md mrshomser.wiki/

# Push
cd mrshomser.wiki
git add .
git commit -m "Add comprehensive documentation"
git push origin master
```
